Voice In
Alert! Unsupported Browser
The speech-to-text APIs needed by Voice In are only available in Google Chrome and Microsoft Edge, the two browsers we support. These are not available on other browsers like Brave, Vivaldi, and other Chromium-based browsers.
We recommend users use Chrome/Edge with the uBlock extension which provides similar privacy protection as these other browsers.
Unfortunately there is a bug in Microsoft Edge for Apple ARM chip (M1/M2/M3/M4) that prevents speech recognition from starting. We have let the Edge devs know about it, however, there is no ETA on when the fix will land. Voice In works fine on Edge for Intel.

Please use Voice In on Google Chrome on your Mac for now.
