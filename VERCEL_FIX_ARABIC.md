### الخطوة 2️⃣: اختر مشروعك
ابحث عن مشروع: **prince-shop47**

### الخطوة 3️⃣: اذهب للإعدادات
```
اضغط على تبويب: Settings (في الأعلى)
ثم اختر: General (من القائمة الجانبية)
```

### الخطوة 4️⃣: ابحث عن "Root Directory"
```
انزل للأسفل حتى تجد قسم:
🗂️ Root Directory

ستجد مكتوب:

              ⬆️ هذا خطأ!
```

### الخطوة 5️⃣: غيّر المجلد
```
1. اضغط على زر "Edit" (تعديل)
2. احذف النقطة (.)
3. اكتب بالضبط: frontend
4. اضغط "Save" (حفظ)
```

**يجب أن يصبح:**
```

```

### الخطوة 6️⃣: أعد النشر
```
1. اضغط على تبويب "Deployments" (في الأعلى)
2. ستجد آخر نشر (Latest Deployment)
3. اضغط على النقاط الثلاث (⋮) بجانبه
4. اختر "Redeploy" (إعادة النشر)
5. اضغط "Redeploy" مرة أخرى للتأكيد
```

### الخطوة 7️⃣: انتظر دقيقة واحدة
سيظهر:
```

✅ Ready       (جاهز!)
```

### الخطوة 8️⃣: جرّب موقعك
افتح: https://prince-shop47.vercel.app/

**يجب أن ترى واجهة المنتجات بالعربية! 🎉**

## 📞 إذا واجهت أي مشكلة:

1. **تأكد من الإعدادات**:
   - Root Directory = `frontend` (ليس `.` ولا فارغ)
   
2. **امسح الـ Cache**:
   - Settings → General → انزل للأسفل
   - اضغط "Clear Cache"
   - ثم Redeploy

3. **تحقق من Logs**:
   - Deployments → اضغط على آخر نشر
   - شاهد الـ Build Logs
   - إذا كان فيه خطأ، أرسله لي
