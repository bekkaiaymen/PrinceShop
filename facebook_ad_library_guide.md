# دليل استخدام Facebook Ad Library + Google Sheets

## مصدر البيانات الرسمي المجاني 100%

**Facebook Ad Library** هو المصدر الوحيد الرسمي والمجاني:
```
https://www.facebook.com/ads/library/
```

## الخطوات العملية (15 دقيقة setup)

### 1. إعداد Google Sheets

أنشئ جدول جديد بهذه الأعمدة:

| A: رابط الإعلان | B: إعجابات | C: تعليقات | D: النسبة % | E: الحكم | F: جودة التعليقات |
|------------------|-------------|-------------|-------------|----------|-------------------|

### 2. الصيغ التلقائية

انسخ هذه الصيغ:

**في خلية D2:**
```
=IF(B2>0,C2/B2*100,"-")
```

**في خلية E2:**
```
=IF(D2>=10,"✅ ناجح","❌ ضعيف")
```

**في خلية F2:**
```
=IF(OR(COUNTIF(C:C,"*سعر*"),COUNTIF(C:C,"*شحن*"),COUNTIF(C:C,"*مقاس*"))>0,"جودة عالية","جودة منخفضة")
```

### 3. سير العمل اليومي (5 دقائق)

1. افتح [Facebook Ad Library](https://www.facebook.com/ads/library/)
2. ابحث عن منتجك (مثل "ساعة رولكس الجزائر")
3. اضغط على إعلان → "عرض التفاصيل"
4. انسخ:
   - رابط الإعلان → عمود A
   - عدد الإعجابات → عمود B
   - عدد التعليقات → عمود C
5. الصيغ ستحسب D, E, F تلقائياً ✅

### 4. تقرير أسبوعي تلقائي

أنشئ Sheet ثاني واستخدم:
```
=QUERY(Sheet1!A:F,"SELECT A,C,D,E WHERE E='✅ ناجح' ORDER BY D DESC")
```
