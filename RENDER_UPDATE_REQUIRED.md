## الخطوة التالية المطلوبة:

### تحديث المتغير البيئي على Render:

1. اذهب إلى: https://dashboard.render.com/
2. سجل الدخول
3. اختر خدمة `princeshop-backend`
4. من القائمة الجانبية، اضغط على **Environment**
5. ابحث عن `FRONTEND_URL`
6. غيّر القيمة من:
   ```
   http://192.168.1.8:3000
   ```
   إلى:
   ```
   https://prince-shop47.netlify.app
   ```
7. اضغط **Save Changes**
8. سيتم إعادة تشغيل الخدمة تلقائياً

## بعد التحديث:

روابط التسويق ستصبح:
```
https://prince-shop47.netlify.app/landing/[PRODUCT_ID]?ref=[AFFILIATE_CODE]
```

## التحقق:

بعد تحديث Render:
1. سجل دخول كمسوق
2. اذهب إلى قسم "المنتجات"
3. اضغط "نسخ رابط التسويق"
4. الصق الرابط - يجب أن يبدأ بـ `https://prince-shop47.netlify.app`
